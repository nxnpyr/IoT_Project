🌾 Smart Rice Bin (ระบบจัดการถังข้าวอัจฉริยะ)
Smart Rice Bin เป็นโปรเจกต์ IoT แบบ Full-stack ที่ช่วยในการบริหารจัดการและดูแลรักษาคุณภาพข้าวสารภายในถังเก็บ โดยรองรับการตรวจสอบสภาพแวดล้อมแบบ Real-time และการสั่งการระยะไกลผ่านเว็บแอปพลิเคชัน

🚀 ฟีเจอร์หลัก (Key Features)
ระบบตรวจสอบสภาพแวดล้อม Real-time: ตรวจสอบอุณหภูมิและความชื้นภายในถัง

ระบบแจ้งเตือนอัจฉริยะ: แจ้งเตือนเมื่อความชื้นสูงเกินเกณฑ์ (70%) พร้อมคำแนะนำในการรักษาคุณภาพข้าว

การควบคุมระยะไกล: สั่งจ่ายข้าวสารตามเวลาที่กำหนด (10, 20, 30 วินาที) และควบคุมระบบระบายอากาศ (พัดลม)

การจัดการข้อมูล: บันทึกวันที่เติมข้าวล่าสุด และระบบรายงานสถิติย้อนหลังในรูปแบบกราฟและตาราง

ความปลอดภัย: ระบบสมาชิกพร้อมการยืนยันตัวตนด้วยรหัส OTP และการกู้คืนรหัสผ่านด้วย Serial Number

โหมดการแสดงผล: รองรับทั้ง Light Mode และ Dark Mode เพื่อความสบายตา

🛠 เทคโนโลยีที่ใช้ (Tech Stack)
Frontend: HTML5, CSS3 (Custom Modern UI), JavaScript (ES6)

Backend: Node.js / EJS (Template Engine)

Database & Real-time: Firebase Realtime Database

Hardware: ESP8266 / Arduino, DHT22 Sensor (สำหรับวัดอุณหภูมิ/ความชื้น), Servo/Relay (สำหรับระบบจ่ายข้าวและพัดลม)


📸 ภาพตัวอย่างหน้าจอ (Screenshots)
หน้า Dashboard (โหมดมืด)
ระบบแสดงผลค่าเซนเซอร์และการควบคุมพัดลม/จ่ายข้าวที่ออกแบบมาให้มองเห็นชัดเจนในโหมดมืด

หน้ารายงานสถิติ (Reports)
แสดงกราฟแนวโน้มความชื้นและอุณหภูมิ พร้อมตารางบันทึกข้อมูลย้อนหลังที่มี Badge สถานะพัดลม
