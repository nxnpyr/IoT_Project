# 🌾 Smart Rice Bin (ระบบจัดการถังข้าวอัจฉริยะ)

> **Smart Rice Bin** เป็นโปรเจกต์ IoT แบบ Full-stack ที่ช่วยในการบริหารจัดการและดูแลรักษาคุณภาพข้าวสารภายในถังเก็บ โดยรองรับการตรวจสอบสภาพแวดล้อมแบบ Real-time และการสั่งการระยะไกลผ่านเว็บแอปพลิเคชัน

## 🚀 ฟีเจอร์หลัก

### 📊 ระบบตรวจสอบสภาพแวดล้อม Real-time
ตรวจสอบอุณหภูมิและความชื้นภายในถังเก็บข้าวอย่างต่อเนื่อง

### 🔔 ระบบแจ้งเตือนอัจฉริยะ
แจ้งเตือนเมื่อความชื้นสูงเกินเกณฑ์ (70%) พร้อมคำแนะนำในการรักษาคุณภาพข้าว

### 📡 การควบคุมระยะไกล
- สั่งจ่ายข้าวสารตามเวลาที่กำหนด (10, 20, 30 วินาที)
- ควบคุมระบบระบายอากาศ (พัดลม)

### 📈 การจัดการข้อมูล
- บันทึกวันที่เติมข้าวล่าสุด
- ระบบรายงานสถิติย้อนหลังในรูปแบบกราฟและตาราง

### 🔐 ความปลอดภัย
- ระบบสมาชิกพร้อมการยืนยันตัวตนด้วยรหัส OTP
- การกู้คืนรหัสผ่านด้วย Serial Number

### 🎨 โหมดการแสดงผล
รองรับทั้ง Light Mode และ Dark Mode เพื่อความสบายตา

## 💻 การติดตั้ง (Installation)

### ข้อกำหนดเบื้องต้น (Prerequisites)
- Node.js (v14 หรือสูงกว่า)
- npm
- Firebase Account

### ขั้นตอนการติดตั้ง

1. **โคลนรีโปสิทอรี่**
   ```bash
   git clone https://github.com/nxnpyr/IoT_Project.git
   cd IoT_Project
   ```

2. **ติดตั้งแพ็คเกจ npm ที่จำเป็น**
   ```bash
   npm init -y
   ```

3. **ติดตั้งแพ็คเกจ Express และ EJS**
   ```bash
   npm install express ejs
   ```

4. **ติดตั้งแพ็คเกจสำหรับ Session และ Flash Messages**
   ```bash
   npm install express-session connect-flash method-override
   ```

5. **ติดตั้งแพ็คเกจอื่นๆ**
   ```bash
   npm install dotenv nodemon --save-dev
   ```

6. **สร้างไฟล์ .env และตั้งค่า Firebase**
   ```bash
   cp .env.example .env
   # แก้ไขไฟล์ .env ด้วยข้อมูล Firebase ของคุณ
   ```

7. **เริ่มการทำงาน**
   ```bash
   npm start
   ```
